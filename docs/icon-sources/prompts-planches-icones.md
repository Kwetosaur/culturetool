# Prompts — Planches d'icônes

Un prompt = un sous-groupe géographique = une génération. Uniquement les icônes marquées
"à venir" dans `plan-carte-icones.md` (les ✅ déjà faites ne sont pas régénérées).
Copie-colle un bloc entier tel quel dans ChatGPT ou Gemini.

**Rappel workflow :**
- Reste dans la même conversation d'un prompt à l'autre (aide le modèle à garder le style).
- Si l'outil le permet, réinjecte une icône déjà validée comme image de référence.
- La couleur d'anneau par catégorie (or/vert/rouge/bleu) n'est **jamais** dans le prompt —
  elle s'ajoute en CSS/SVG après coup. Tous les prompts ci-dessous sont volontairement
  monochromes, quelle que soit la catégorie.
